# 💱 Конвертер валют (Telegram бот)

Простой бот с кнопками. Переводит **USD**, **EUR** и **RUB**.

---

## 🕹️ Команды

| Команда | Что делает |
|---------|-------------|
| `/start` | Начать конвертацию |
| `/values` | Показать доступные валюты |
| `/help` | Помощь |

---

## 🚀 Как пользоваться

1. Нажми `/start`
2. Выбери **валюту, которую хочешь перевести** (USD / EUR / RUB)
3. Выбери **валюту, в которую перевести**
4. Введи **число** (например, `100`)

Бот покажет результат:  
👉 `100 USD = 8500.00 RUB`
